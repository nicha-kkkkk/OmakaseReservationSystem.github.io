
# 🍣 Kitsune Omakase Reservation System

A web-based reservation system for Omakase restaurants that allows customers to reserve courses, customize menus, submit allergy requests, and complete payment confirmation. The system also includes an admin panel for managing courses, menus, and reservations.

---

## 📖 Overview

This project was developed as part of a university course.

The system provides:

- 🍣 Omakase course reservation
- 📅 Reservation date & time selection
- 📝 Food allergy notification
- ✨ Menu customization
- 💳 Payment confirmation
- 👨‍💼 Admin management
- 📊 Reservation management

---

## ✨ Features

### Customer

- Register / Login
- Browse Omakase courses
- Select reservation date and time
- Food allergy notification
- Menu customization
- Payment confirmation
- Reservation history

### Administrator

- Dashboard
- Manage courses
- Manage menu items
- Manage reservations
- Confirm payments
- Update seasonal menus
